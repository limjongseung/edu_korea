event :

웹페이지가 응답할수 있는 모든 방문자의 행동을 이벤트라 합니다

이벤트는 어떤일이 발생하는 정확한 순간을 나타냅니다
- 요소위로 마우스 이동
- 라디오버튼 선택
- 요소클릭

dom(document object model) : 웹페이지에대한 인터페이스를 의미합니다
dom event
- mouse event
1)click 
2)dblclick 
3)mouseenter
4)mouseleave

- keyboard event
1)keypress
2)keydown
3)keyup

- form event
1) submit : 전송
2) change : 변경
3) focus : 초점이 맞을때
4) blur : 흐릿하게 해줍니다

- document/window event
1)load : 윈도우나 문서가 로드될때
2)resize : 사이즈 변경
3)scroll : 스크롤바 관련 
4)unload : 로드하지 않을때

더블클릭

전 더블클릭해야 사라집니다

마우스가 들어가면 이벤트 실행

여기에 마우스를 올려보세요

마우스가 들어갔다가 나오면 실행

마우스 들어왔다가 나와주세요
