jquery

    html,css,자바스크립트에 대한 기본이해가 필요합니다
    jquery는 무엇인가요?
    - 경량의 적게 작성하고 더많이 수행하는 라이브러리 입니다
    - 자바스크립트에서 사용하는걸 좀더 쉽게 사용할수 있도록 하는것 입니다
    - 동적으로 움직이는 어떤 개발언어서도 jquery를 사용합니다
    - cdn선언을 해야합니다
    - 부트스트랩과 마찬가지로 cdn선언이 안되면 작동하지 않습니다

    구문 : HTML요소를 선택하고 요소에 대해 일부작업을 수행하기 위해 제작되었습니다
    기본구문은 $(선택자)입니다.action()
    - 정의/엑세스를 위한 $기호
    예시)
    $(this).hide() : 현재요소를 숨깁니다
    $("p").hide() : 모든 
요소를 숨깁니다
    $(".test").hide() : class test인 모든 요소를 숨깁니다
    $("#test").hide() : id test인 요소를 숨깁니다

    Document Ready Event : 문서준비이벤트
    $(document).ready(function(){
    jquery methods
    });
    문서로드가 완료되기전에 jquery코드가 실행되는걸 방지하기 위한것입니다
