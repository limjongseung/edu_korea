concat() : 배열을 결합하고 결합된 배열이 있는 배열을 리턴
        문법 : array1.concat(array2,....);
        

        
        2)constructor : 배열생성자를 가져옴
        - 자바스크립트 배열은 constructor는 
        => function Array{[네이티브코드]}
        
        
        
        3)copyWithin() : 배열요소를 배열의 다른 위치에 복사합니다. 
        문법 : array.copyWithin(target, start, end)
        

        
        4) entries() : 메서드는 키/값 쌍으로 array iterator객체를 리턴
        

        
        
        5) every : 함수에 대한 값이 참이냐 거짓이냐
        예시 ) 나이에 모든 값이 18세 이상인지 확인합니다.
        

        
        6)fill() : 배열에 지정된 요소를 값으로 채웁니다.
        -원래 배열을 덮어씁니다.
        - 문법 : array.fill("kiwi",2,4);
        [banana,orange,apple,manggo] -> [0,1,2,3]
        
        
    
        

        7) filter() : 함수에서 제공하는 데스트를 통과하는 요소로 채워진 새배열을 만듭니다 원래배열을 변경하지 않습니다.
