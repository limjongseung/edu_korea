Booleans
    - 두값중에 하나를 가질수 있습니다.
    true, false  참이냐 거짓이냐를 확인해줍니다.
    

    

    Boolean 메서드 및 속성
    - constructor : Javascript의 Boolean 프로토타입을 생성한 함수를 반환합니다.
    function Boolean(){[네이티브 코드]}
    - prototype :  프로토타입에 속성 및 메서드를 추가할수 있습니다.
    

    
    - toString() : Boolean값을 문자열로 변환하고 결과를 리턴
    Boolean에 형태가 string 
    
    - valueOf() : Boolean에 기본값을 리턴
