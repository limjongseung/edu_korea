메타문자
    1). : that's hot!/h.t/
    지정된 문자에 시작과 끝으로 끝나는 워딩이 검색
    

    
    2)/w : 단어문자에 대한 전역검색
    - 밑줄을 포함하는 문자 az,AZ,0-9입니다.
    - 특수문자 제외
    

    
    2)\W : 비단어문자에 대한 전역검색
    - 밑줄을 포함하는 문자 az,AZ,0-9입니다.
    - 특수문자 제외
    

    
    4)\d : 0에서 9까지의 숫자에 대한 전역검색
    

    
    6)\s : 공백문자에 대한전역검색
    7)\S : 공백이 아닌 문자에 대한 전역검색
    8)\b : 지정한 문자에 대한 인덱싱 번호 리턴
    

    
    9) \B : 지정한 문자가 아닌 첫번째 항목에 인덱싱번호를 리턴
    수량자 : 갯수를 파악
     + : 지정된 문자에 전체 갯수를 파악
     

     
     2) * : 지정된 문자에 뒤따르는 문자에 대한 검색
     

     
     3) ? : 1 다음에 0 또는 하나의 0문자가 오는 전역검색을 실행
