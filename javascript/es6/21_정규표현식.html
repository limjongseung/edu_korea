검색및 바꾸기 기능을 수행하는데 사용합니다.
    1) 수식어(modifiers) : 수정자라고도 하며 대소문자를 구분하지 않고 
    전역검색을 수행하는 데 사용됩니다
    
    g : 전역일치(한정자) 모든 일치 항목을 찾아줌
    

    
    i : 대소문자를 구분하지 않고 검색합니다
    m : 이수정자는 여러줄에 대한 일치를 검색합니다.
    - 시작 ^ 종료 $의 동작에만 영향을 미칩니다.
    

    
    2)브라켓 : 대괄호는 문자범위를 찾는데 사용
    [abc]: 문자 a,b 또는 c
    

    
    

    
    [^abc] : 문자a,b 또는 c 가 아님
    - 지정된 문자를 제외한 공백까지 검색
    

    
    [0-9] : 0-9까지의 숫자 또는 숫자 범위
    [^0-9] : 숫자가 아닌 문자
    

    
    (x|y) : (red|green) : 모든 문자가 될수 있으며 지정된 대안을 찾는데 사용됩니다.
