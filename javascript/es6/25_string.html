문자열

    자바스크립트 문자열은  ""와 같은 일련의 문자를 저장합니다.
    문자열 인덱스는 0부터 시작합니다.
    

    
    -문자열 메서드
    1)charAt(): 문자열에 지정된 인덱스(위치)에 있는 문자를 리턴
    -1을 붙이면 마지막 문wk 리턴
    

    
    -마지막 문자를 가져올 경우 - 
    

    
    2)charCodeAt(): 지정된 문자의 unicode를 리턴
    -마지막 문자의 워드를 리턴할때는 -1을 사용하세요
    -비슷한 경우 => codePointAt()
    

    
    3)concat(): 두개 이상의 문자열을 결합합니다.
    - 기존 문자열을 변경하지 않습니다.
    - 새문자열을 리턴합니다. 
    

    
    4)endsWidth(): 지정한 문자로 끝나면 true 아니면 flase
    -대소문자를 구분합니다.
    

    
    5)fromCharCode(): 유니코드값을 문자열로 리턴
    

    
    6)includes(): 지정된 문자열이 포함되어 있는지
