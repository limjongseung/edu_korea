16) lastIndexOf() : 지정된 값의 마지막 인덱스(위치)를 리턴 
  값을 찾을 수 없는 경우 -1를 리턴
  17) length() : 배열의 길이를 리턴 : 배열에서 문자열이 4개 있으면 4로 리턴
  

  
  18) map() : 배열요소에 함수를 호출해서 새배열을 리턴
  

  
  19) pop() : 마지막 요소를 제거
  

  
  20) prototype : 배열에 새속성과 메서드를 추가
  

  
  21) push : 배열에 끝에 새항목을 추가합니다. 
  

  
  22) reduce() : 배열요소에 대한 감속기능(함수의 누적결과라는 단일값을 리턴)
  

  
  23) reduceRight() : 배열요소에 대한 감속기함수 방법은 오른쪽에서 왼쪽으로 작동
  24) reverse() : 배열의 요소순서를 반대로 바꿉니다. 
  25) shift() : 배열의 첫번째 항목을 제거합니다.
  26) slice() : 배열에서 지정된 범위만큼 추출해서 새배열로 만들어 줍니다.
  27) some() : 배열요소가 테스트를 통과하는지 확인하고 (콜백함수로 제공됩니다.)
  28) sort() : 배열의 요소를 정렬 원래배열을 덮어씁니다.
  29) splice() : 배열요소를 추가 및 제거합니다.
  30) toString() : 쉼표로 구분된 배열값이 있는 문자열로 리턴
  31) unshift() : 배열의 시작부분에 새요소를 추가합니다.
  32) valueOf() : 배열자체를 리턴(똑같이 나온다고 보면 됩니다.)
