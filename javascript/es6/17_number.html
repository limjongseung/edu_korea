Number

    - 정수
    - 음수
    - 둘다포함하여 소수점
    - 지수

    - 숫자메서드-
    1) isFinite() : 숫자가 유효한지 아닌지 : true, false
    2) isInteger() : 정수인지 아닌지 : true,false
    3) isNaN() : 넘버인지 아닌지 : true, false
    4) isSafeInteger : 안전한 정수
    5)MAX_SAGE_INTEGER : 최대 안전정수(2 53 -1)
    6)MIN_SAGE_INTEGER : 최소 안전정수(2 53 -1)
    7)MAX_VALUE : 자바스크립트에서 표현할 수 있는 가장 큰숫자
    

    
    8)MIN_VALUE : 자바스크립트에서 가능한 가장 작은 숫자
    

    
    9)NaN : 숫자가 아님을 표시
    10)NEGATIVE_INFINITY : 음의 무한대
    11)POSITIVE_INFINITY : 양의 무한대
    12)parseFloat: 문자열로 구문 분석하고 첫번째 숫자를 리턴
    13)parseInt():문자열로 구문 분석하고 첫번째 정수를 리턴
    14)toExponential: 숫자를 지수 표기법으로 리턴
    

    
    15)toFixed(): 문자열을 지정된 소수 자리수로 반올림합니다.
    

    
    16)toLocaleString(): 현지 언어 형식을 사용하여 숫자를 문자열로 리턴
    

    
    17)toPrecision() : 숫자를 지정된 길이로 포멧
    

    
    18)toString(): 숫자를 문자열로 변환
    19)valueOf() : 숫자의 기본 값을 가져옵니다.
