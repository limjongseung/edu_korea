- Math 개체 메서드 및 속성 -
1)Math.abs() : 숫자의 절대값(음수가 아닌)을 리턴
- 숫자가 아닌경우 NaN
- null 값을 0으로 리턴





2)acos() : 숫자의 아크코싸인(라디안)을 리턴
- 0과 PI사이의 값을 리턴
- -1에서 1사이의 매개변수를 리턴
- Math.acos(-1) : PI를 리턴
- Math.acos(0) : PI/2 를 리턴
- Math.acos(1) : 0 를 리턴
- -1 ~ 1사이를 벗어나거나 숫자가 아닌경우 NaN





3)acosh() : 숫자의 쌍곡선 역코사인을 리턴
- 매개변수가 1보다 작으면 매서드는 NaN을 리턴




4)asin() : 숫자의 아크사인(라디안을)리턴
- -PI/2와 PI/2 사이의 값을 반환합니다.
- -1에서 1 사이의 매개변수를 리턴
- Math.asin(-1)-PI/2를 반환합니다.
- Math.asin(0)0를 반환합니다.
- Math.asin(1)PI/2를 반환합니다.





5)asinh() : 쌍곡선 아크사인에 숫자를 리턴
- 0일때 0나오고
- 음수 양수모두되며 값에 제한이 없음




6)atan() : 숫자의 아크탄젠트를 -PI/2와 PI/2 라디안 사이의 값으로 반환합니다.



7)atan2() : 인수 몫의 아크탄젠트를 PI와 -PI라디안 사이의 숫자값으로 리턴
반환된 숫자는 양의x축과 점(x,y)사이의 시계 반대방향 각도(도가 아님)을 나타냅니다
문법-atan(y,x)
